Weighted Tardiness Scheduling with Sequence-Dependent Setups

This set of benchmark instances for the problem known as Weighted Tardiness Scheduling with Sequence Dependent Setups originated in 2003 as part of the Ph.D. Dissertation research of Vincent A. Cicirello from Carnegie Mellon University's Robotics Institute. This benchmark set includes the instances, the current best known solutions, a problem generator, and links to relevant publications.
